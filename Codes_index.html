Use these reminders as a fast rehab checklist for blighted housing work, then scroll the embedded 2024 International Building Code reference below.

01

Confirm the scope of work

Separate repair, alteration, and change-of-occupancy decisions early. Blighted house renovations often move beyond simple repair once layout, use, or structural work changes.

02

Rebuild life-safety paths

Check egress width, bedroom escape openings, corridor conditions, and fire-resistance needs wherever the existing shell has been compromised or re-partitioned.

03

Plan upgrades as one system

Structural repairs, accessibility moves, and envelope replacement should be coordinated together so the renovation strategy remains buildable and code-aligned from schematic through permit.
